# Lat OS

AI-powered voice assistant with local screen automation for Android.

## Features

- **Voice Control**: Speak commands to control your phone
- **Local Processing**: Screen data never leaves your device
- **Deep Link Routing**: Fast app launching via intents
- **Accessibility Automation**: Control any app via Android Accessibility Service
- **Vision Fallback**: AI-powered screen understanding when accessibility tree is empty
- **Privacy-First Search**: Online search only when explicitly requested

## Permissions Required

| Permission | Why | Required For |
|---|---|---|
| Microphone | Voice command input | Core functionality |
| Accessibility | Screen reading and control | Core functionality |
| Overlay | Floating confirmation UI | User experience |
| Internet | Web search only | Search feature |
| Notifications | Background service status | Background operation |
